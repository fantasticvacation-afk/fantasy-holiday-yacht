## 🔧 技术细节

### i18n 键命名规范

**全局命名空间**（组件级 UI）:
- `nav.*` - 导航栏
- `footer.*` - 页脚
- `_ph.*` - 占位符文本

**页面命名空间**（页面特定内容）:
- `index.N` - 首页
- `about.N` - 关于我们
- `contact.N` - 联系我们
- 等等

### 响应式断点

```css
/* 平板端 */
@media(max-width:1024px) { /* ... */ }

/* 大屏手机 / 小平板 */
@media(max-width:768px) { /* ... */ }

/* 大屏手机 */
@media(max-width:480px) { /* ... */ }

/* 小屏手机 */
@media(max-width:375px) { /* ... */ }

/* 超小屏手机 */
@media(max-width:320px) { /* ... */ }
```
